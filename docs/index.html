Model Architecture

Input (64×64×1 grayscale) │ ├─ Conv2D(32, 3×3, ReLU) → MaxPool(2×2) ├─ Conv2D(64, 3×3, ReLU) → MaxPool(2×2) ├─ Conv2D(128, 3×3, ReLU) → MaxPool(2×2) │ [Deep model adds Conv2D(256) block] │ ├─ Flatten → Dense(128, ReLU) → Dropout └─ Dense(1, Sigmoid) → P(drowsy)

OptimizerAdam (lr = 1e-3)
